# Cafe Finder

A fully free Cafe Finder built with Python, Streamlit, and OpenStreetMap.  

**Features:**
- Auto-detects user location
- Finds cafes within a selectable radius
- Sorts cafes by distance
- Shows an interactive map with markers
- Clickable directions to cafes

**How to run locally:**
```bash
pip install -r requirements.txt
streamlit run cafe_finder.py
